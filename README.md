# 🎨 CollaboDRAW

> A real-time collaborative whiteboard application for seamless multi-user drawing.
  [![CollaboDRAW Demo](https://img.youtube.com/vi/t0_OSyVOm5k/maxresdefault.jpg)](https://youtu.be/t0_OSyVOm5k)

## 📝 About The Project

CollaboDRAW is a shared digital canvas that allows multiple users to draw, sketch, and brainstorm together in real-time. Designed with a focus on low latency and a clean user interface, it synchronizes brush strokes across all connected clients instantly.

**Key Features:**
* ⚡ **Real-Time Synchronization:** See others draw instantly without refreshing.
* 🖌️ **Interactive Tools:** Pen, eraser, color picker, and dynamic brush sizing.
* 🧹 **Clear Canvas:** One-click reset for all connected users.
* 📱 **Responsive UI:** Clean, distraction-free environment.

## 🛠️ Built With

* **Frontend:** HTML5 Canvas API, CSS3, JavaScript
* **Backend:** Node.js, Express.js
* **Real-Time Engine:** Socket.io (WebSockets)
